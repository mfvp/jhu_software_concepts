Module 1: Personal Website

Project Description

This assignment teaches the fundamentals of website construction. The goal is to build a personal developer website that includes a biography, contact information and links, and details about Python projects. It provides an opportunity to be creative while learning the fundamentals of web development. The website will be updated throughout the course as the set of Python projects grows.

The project uses Flask, a micro web framework written in Python. It covers working with HTML to structure content, and CSS (Cascading Style Sheets) to control how HTML elements are displayed. The output is a functional personal website with details about the developer, their projects, and contact information.

Skills

  • Flask: Python micro web framework for routing and serving pages
  • HTML: structuring web page content
  • CSS: styling and formatting HTML elements
  • Searching Methods: techniques for locating and retrieving data
  • Data Cleaning: processing and preparing raw data for use
  • JSON Data Object Storage: storing and reading structured data in JSON format

How to Run

Clone the repository, install dependencies with pip install -r requirements.txt, then run python run.py. The site will be available at http://localhost:8080 (open in your browser).
