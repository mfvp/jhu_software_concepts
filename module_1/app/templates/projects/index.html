Module 1: Personal Website

Project Description

This assignment teaches the fundamentals of website construction. The goal is to build a personal developer website that includes a biography, contact information and links, and details about Python projects.

The project uses Flask, a micro web framework written in Python. It covers working with HTML to structure content, and CSS (Cascading Style Sheets) to control how HTML elements are displayed.

Skills

  • Flask: Python micro web framework
  • HTML: structuring web page content
  • CSS: styling and formatting HTML elements

How to Run

Clone the repo, install with pip install -r requirements.txt, then run python run.py.
