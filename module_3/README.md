# Module 3 - PostgreSQL Database and Flask Analysis

JHU Software Concepts in Python — Module 3

## Overview

This module loads scraped Grad Cafe applicant data (from Module 2) into a
PostgreSQL database and provides a Flask web application for interactive analysis.

## Setup

### 1. Install PostgreSQL

Follow the lecture slide instructions for Windows:

1. Download the PostgreSQL installer from https://www.postgresql.org/download/windows/
2. Run the installer. During setup, set a password for the `postgres` user (remember it!).
3. Default port is 5432 — leave that as-is.
4. Once installed, open **pgAdmin** (installed alongside PostgreSQL) and create a new database named `gradcafe`.
   - Right-click "Databases" → Create → Database → name it `gradcafe` → Save.

### 2. Install Python Dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure the Database Connection

Credentials are stored in a `.env` file so they never get committed to git.

Copy the example file and fill in your password:

```bash
copy .env.example .env
```

Then open `.env` and set your PostgreSQL password:

```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=gradcafe
DB_USER=postgres
DB_PASSWORD=your_postgres_password_here
```

> **Note:** `.env` is listed in `.gitignore` and will never be committed.
> If you share this code, the recipient copies `.env.example` to `.env` and sets their own password.

### 4. Load Data into PostgreSQL

Make sure `llm_extend_applicant_data_run.jsonl` is in the `module_3/` folder
(it is generated by the Module 2 LLM enrichment step), then run:

```bash
python load_data.py
```

This creates the `applicants` table and loads all entries into PostgreSQL.

### 5. Run SQL Queries

```bash
python query_data.py
```

This prints the answers to all analysis questions to the console.

### 6. Run the Flask Web App

```bash
python app.py
```

Then open your browser to http://localhost:5000
